# Huanyu Kuntai Website Project

## Brand
- **Company:** Huanyu Kuntai Machinery Co., Ltd.
- **Website:** huanyukuntai.com
- **Type:** B2B exporter of truck parts & construction machinery

## Color Scheme
- Primary: `#1e3a5f` (Deep Navy)
- Accent: `#f97316` (Orange)
- Background: `#f8fafc`
- All colors in tailwind.config.js

## Data
- All product data: `src/data/products.ts`
- No backend/CMS — static JSON data only

## Deploy Notes
- Build: `npm run build`
- Start: `npm start -p 3000`
- Node 18+ required
