# Huanyu Kuntai Website

Professional exporter website for truck parts and construction machinery components.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

## Color Scheme
- Primary: `#1e3a5f` (Deep Navy Blue)
- Accent: `#f97316` (Industrial Orange)
- Background: `#f8fafc` (Cool White)

## Pages
- `/` - Home
- `/products` - Product Catalog
- `/products/[slug]` - Product Detail
- `/applications` - Industry Applications
- `/about` - About Us
- `/contact` - Contact & Inquiry Form

## Notes
- All data is currently static (in `src/data/products.ts`)
- Images use placeholder divs — replace with real product photos
- Contact form is UI demo only — connect to email service for production
